## Installation

### Clone
```bash
$ git clone https://github.com/reihnagm/React-Native-Hiring-Channel-App.git
$ cd React-Native-Hiring-Channel-App
$ npm install or yarn install
```

### Create Environment Variable
```bash
$ cp .env.example .env
$ nano .env
```

### Start Development Server
```bash
$ npm run android
```
